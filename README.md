# 优势跑酷

一个极简横版跑酷小游戏：按空格键跳跃，自动向前奔跑，看看你能取得多少优势。

## 在线游玩

[点击这里直接开始游戏](https://eurus07e.github.io/jjs/?play=1)

## 操作

- 点击“开始”或打开上面的直达链接进入游戏。
- 游戏进行中按 `Space` 跳跃。
- 死亡后点击“重新开始”再来一局。

## 特色

- 经典 2D 跑酷节奏。
- 随分数提升逐渐加速。
- 跳跃、里程碑和死亡都有语音触发。
- 纯静态网页，无需安装依赖。

## 本地运行

```bash
python3 -m http.server 4173
```

然后打开：

```text
http://127.0.0.1:4173/
```
